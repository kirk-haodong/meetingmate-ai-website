3步开始使用

零配置，开箱即用

1

安装 OpenClaw

npm install -g openclaw
2

配置 API 密钥

在 ~/.openclaw/config.json 中配置：

  • Whisper API 密钥（语音识别）
  • MiniMax/OpenAI API 密钥（文本生成）
3

运行 Skill

openclaw run meeting-transcriber --audio meeting.mp3

2分钟后，你将获得：完整会议纪要、待办任务列表、可搜索的会议记录
