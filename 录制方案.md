## 🛠️ 录制工具推荐

### 方案A：本地录制（推荐）

**录屏软件**：
- **Windows**：OBS Studio（免费）或 Bandicam
- **macOS**：QuickTime Player（自带）或 ScreenFlow
- **Linux**：OBS Studio

**音频录制**：
- 使用外置麦克风或耳机麦克风
- 后期可用 Audacity 降噪处理

**剪辑软件**：
- 剪映（免费，简单易用）
- Premiere Pro（专业）
- DaVinci Resolve（免费专业）

---

### 方案B：在线录制

**浏览器扩展**：
- Loom（免费，云端存储）
- Screencastify（Chrome扩展）

## 📤 视频上传

### 推荐平台

| 平台 | 优点 | 注意事项 |
|------|------|----------|
| **Bilibili** | 国内主流，适合技术视频 | 需要注册账号 |
| **YouTube** | 国际平台，画质好 | 需要科学上网 |
| **阿里云盘** | 国内访问快 | 生成分享链接 |
| **GitHub Releases** | 与代码一起发布 | 有文件大小限制 |

### 推荐做法
- 上传到 Bilibili（国内评委方便观看）
- 同时上传一份到 GitHub Releases 作为备份
- 在 README 中添加视频链接
