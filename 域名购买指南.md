## 📝 购买步骤（以阿里云为例）

### 步骤1：查询域名是否可用
1. 访问 https://wanwang.aliyun.com
2. 搜索 "meetingmateai.com"
3. 查看是否可注册

### 步骤2：注册账号并购买
1. 注册阿里云账号（需实名认证）
2. 将域名加入购物车
3. 选择购买年限（推荐2-3年更划算）
4. 支付（支持支付宝/微信）

### 步骤3：DNS 解析配置
购买后，在阿里云控制台添加解析记录：

```
记录类型：A
主机记录：@
解析线路：默认
记录值：43.230.8.188
TTL：10分钟

记录类型：A
主机记录：www
解析线路：默认
记录值：43.230.8.188
TTL：10分钟
```

### 步骤4：等待生效
- 一般 10分钟-2小时 生效
- 最长不超过 48小时

## 🚀 配置 Nginx（购买域名后）

```bash
# 编辑 Nginx 配置
cat > /etc/nginx/sites-available/meetingmate << 'EOF'
server {
    listen 80;
    server_name meetingmateai.com www.meetingmateai.com;
    
    root /root/meetingmate-ai/website;
    index index.html;
    
    location / {
        try_files $uri $uri/ =404;
    }
}
EOF

# 重载配置
nginx -s reload
```

## 🔒 申请 SSL 证书（HTTPS）

### 方式1：Let's Encrypt（免费）

```bash
# 安装 certbot
apt-get install certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d meetingmateai.com -d www.meetingmateai.com

# 自动续期（每3个月）
echo "0 0 1 * * certbot renew --quiet" | crontab -
```

### 方式2：阿里云免费证书
1. 登录阿里云控制台
2. 进入 SSL 证书服务
3. 申请免费 DV 证书
4. 下载 Nginx 版证书
5. 配置到服务器
